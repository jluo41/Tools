# Report Style Profile

Drafting guide for formal stakeholder report artifacts.


## Voice examples

**Executive summary (executive audience):**
```
Bottom line: Timing-aware refill SMS increases 7-day
adherence by 8.2pp (95% CI: 4.1–12.3pp) among high-risk
patients. Projected annual savings: $2.4M across the
current panel. Recommended: expand to all eligible patients
in Q3. (See K03, W02.)
```

**Regulatory finding (regulator audience):**
```
3.2 Effectiveness

The intervention demonstrated a statistically significant
improvement in 7-day refill adherence (OR = 1.42, 95% CI:
1.18–1.71, p < 0.001).¹ The effect was consistent across
age and gender subgroups (Table 2). No adverse events were
reported during the pilot period.

¹ K03: Timing sensitivity in prescription refill behavior.
  Based on probe P.A12, N = 4,230 patients, 2025-Q4 cohort.
```


## Drafting rules

1. Structure depends on audience:
   - Regulator: exec summary → methodology → findings →
     limitations → recommendations → appendix
   - Executive: bottom line → evidence → ask
   - Partner: context → joint findings → next steps

2. Every factual statement must cite a K/W entry.
   - Regulator: footnote (¹ K03_...)
   - Executive: endnote
   - Partner: inline parenthetical

3. Tables and figures with captions required for data claims.

4. Limitations section mandatory for regulator audience.

5. Length per audience:
   - Regulator: ≤ 1500 words
   - Executive: ≤ 600 words
   - Partner: ≤ 800 words


## Audience pairing

```
audience=regulator   → formal, footnoted, comprehensive
audience=executive   → direct, outcome-oriented, brief
audience=partner     → professional, collaborative tone
```


## Self-review checklist

```
[ ] Structure matches audience template
[ ] Every factual claim has citation (format per audience)
[ ] No contested/superseded K/W cited as confirmed
[ ] Length within budget for audience
[ ] Limitations section present (regulator)
[ ] Data tables/figures have captions
[ ] cited_K / cited_W in frontmatter
```
