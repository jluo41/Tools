# Audience Profile: Regulator

## Tone
Formal, neutral. Spelled-out terminology. Comprehensive.

## Constraints
- Length: ≤ 1500 words
- Jargon: spelled out on first use
- Reading level: professional/legal
- Citations: footnote + K-id — "...improvement.^1" + "^1 K03_..."

## Lifecycle mappings

### → Claims
All claims must include statistical evidence. No directional
language without quantification. Confidence intervals required
for primary claims. Limitations section mandatory.

### → Design
Structure: executive summary, methodology, findings, limitations,
recommendations. Appendix for raw data references.

### → Variants
Single comprehensive report per regulatory submission.
No abbreviated variants.

### → Draft
Formal register. No contractions. Define all acronyms.
Every factual statement footnoted.
