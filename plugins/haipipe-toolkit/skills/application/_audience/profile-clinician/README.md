# Audience Profile: Clinician

## Tone
Precise, clinical. Professional but not formal. Evidence-forward.

## Constraints
- Length: ≤ 400 words
- Jargon: expected and appropriate
- Reading level: professional
- Citations: inline K-id mandatory — "(K03)" or "(see K03)"

## Lifecycle mappings

### → Claims
Claims must cite specific evidence with effect sizes.
Confidence intervals or directional language required.
"Statistically significant" only with p-value or CI.

### → Design
Lead with clinical relevance. Include patient segment
specifics. Reference the evidence base concisely.

### → Variants
May include dashboard reports, clinical summaries, or
in-workflow alerts alongside direct messages.

### → Draft
Imitate the style in exemplars/. Direct opening with
clinical context, evidence-backed recommendation, specific
patient criteria, expected outcome.
