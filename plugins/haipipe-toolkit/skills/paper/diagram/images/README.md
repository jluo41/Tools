# Paper Skill Diagrams

Generated visual diagrams for the HAI-Pipe paper skill mental model.

| File | Purpose |
|------|---------|
| `paper-pitch-layer.png` | Shows `0-pitch/` as the one-minute story and provenance layer. |
| `paper-lifecycle-layered-convergence.png` | Shows the paper lifecycle as layered convergence from seed to present. |
| `paper-loopback-diagnosis.png` | Shows review-gate loopbacks to edit, plan, architecture, narrative, pitch, or seed. |
| `research-stack-paper-lifecycle.png` | Shows how upstream narrative/probe/task/discovery work feeds the paper lifecycle. |
| `aris-vs-haipipe-workflow.png` | Compares ARIS forward workflow with HAI-Pipe paper story-state evolution. |

Original generated images were created outside the repo under `.codex/generated_images/`.
## Stage 0

- `stage0-topic-to-paper-prospectus.svg` — first-scene diagram for the paper
  lifecycle walkthrough: project seed, paper prospectus folder, narrative handoff,
  discovery loop, and promotion gate.
- `stage0-topic-to-paper-prospectus.png` — rendered PNG copy of the same diagram.
- `stage0-stage1-project-lifecycle.svg` — explains why Stage 0 can move quickly
  to Stage 1, while the heavy task/probe/insight/narrative loop happens after
  the paper prospectus folder exists.
- `stage0-stage1-project-lifecycle.png` — rendered PNG copy of the same diagram.
