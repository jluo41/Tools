Sentences in draft text must carry one idea each. Split any sentence over ~25 words that bundles two things (claim + explanation, list + because clause, definition + application). User flagged multiple instances: "this sentence is too long", "very hard to read", "not sure what you want to say." The fix is always the same: split into two sentences. Applies to both outline (.md) and tex. Run a word-count audit after drafting to catch violations before the user reads.
