# Clinical Exemplars - content to imitate

Store the CONTENT of similar clinical papers here (PDF, or extracted .md/.txt), so
the writing stages can imitate their **language style and preferences**. This is
NOT a citation list (that is `../references/README.md`); it is a style corpus.

## How to add

- Drop a PDF: `exemplars/<slug>.pdf` (e.g. `barnett-2017-nejm-opioid-prescriber.pdf`).
- Or store extracted text: `exemplars/<slug>.md` (the structured abstract +
  principal-findings paragraph + a few Methods and Discussion paragraphs are enough
  to capture style).
- Prefer open-access or author-provided PDFs.

## What to capture per exemplar (feeds `../style-profile.md`)

- Structured-abstract arc and exact phrasing (Importance ... Conclusions and Relevance).
- Associational result sentences ("X was associated with Y (95% CI)").
- The Discussion-opening principal-findings sentence.
- Discussion structure (principal findings -> prior work -> mechanism/trade-off ->
  limitations -> conclusion).
- Tone, sentence length, jargon level, how it frames clinician behavior without blame.

## Candidate topics to source (similar to this paper)

observational cohort studies on prescribing variation and patient safety in the
paper's domain; physician-characteristic-to-prescribing studies; overuse /
de-implementation and vulnerable-population work; trial exemplars if relevant.
