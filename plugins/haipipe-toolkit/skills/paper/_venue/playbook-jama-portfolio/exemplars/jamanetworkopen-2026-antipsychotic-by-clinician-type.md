# Exemplar: Kim 2026 (JAMA Network Open) - antipsychotic prescribing by clinician type

JAMA-family style exemplar (JAMA Network Open), Original Investigation. Clinician
characteristic (clinician TYPE) as the explanatory variable for prescribing in
Medicare Part D claims - the same "clinician attribute -> prescribing" shape as ProjB.
Full text stored as `jamanetworkopen-2026-antipsychotic-by-clinician-type.xml` (no
render-PDF exists yet for 2026 articles; this is the real full text in JATS XML).

## Citation

Kim Y, Zhou X, Du S, et al. Shifts in Antipsychotic Prescribing by Clinician Type for
Medicare Part D Beneficiaries, 2013-2023. JAMA Netw Open. 2026;9. e26.3410.
doi:10.1001/jamanetworkopen.2026.3410. PMC13019238. Open access.

## Type and why it fits

- Article type: **Original Investigation** (full article).
- Clinician-type exposure -> prescribing outcome, national Medicare Part D, repeated
  cross-sectional. Mirrors ProjB's "physician attribute -> opioid prescribing in
  claims" design.

## Style elements to mirror

- JAMA Network Open **Key Points box** (Question / Findings / Meaning) above the
  abstract: "QuestionHow have patterns of antipsychotic prescribing by clinician type
  changed among Medicare Part D beneficiaries from 2013 to 2023?"
- Findings stated as shares with explicit numbers: "the proportion ... increased from
  13.8% to 39.6%".
- Structured abstract + associational, cross-sectional voice.
