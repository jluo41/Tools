# Grant Playbook - Background References

Verified, real prior work that this proposal can cite in its background / related
work and significance sections. Each entry says what we borrow: the gap framing, a
preliminary-data anchor, or a competing-funded-project to position against.

RULE: store ONLY real, verifiable papers and funded projects (DOI / award number
where possible). Never fabricate a citation, a grant, or a PI credential. Re-verify
with `citation-audit` before any of these enters the proposal bibliography.

## Search categories

1. Foundational prior work in the proposal's domain (what the aims build on).
2. Recent work that establishes the gap and the "why now".
3. Competing FUNDED projects in the same area (KAKEN, NSF Award Search, NSFC
   databases) to position against and show field awareness.
