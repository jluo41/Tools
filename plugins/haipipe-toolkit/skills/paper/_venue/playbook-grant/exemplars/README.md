# Grant Exemplars - content to imitate

Store the CONTENT of similar FUNDED proposals here (PDF, or extracted .md/.txt), so
the writing stages can imitate their **persuasive style and reviewer preferences**.
This is NOT a citation list (that is `../references/README.md`); it is a style
corpus. Prefer proposals that actually got funded by the same agency and program.

## How to add

- Drop a PDF: `exemplars/<slug>.pdf` (e.g. `nsf-2022-career-funded.pdf`).
- Or store extracted text: `exemplars/<slug>.md` (the Specific Aims / 概要 plus the
  significance and feasibility paragraphs are enough to capture style).
- Prefer publicly released or author-provided funded proposals.

## What to capture per exemplar (feeds `../style-profile.md`)

- Specific Aims / summary arc and exact phrasing.
- How the gap and significance are stated in one paragraph.
- Aim phrasing (rationale -> objective -> deliverable, with risk and fallback).
- Feasibility structure (preliminary data -> timeline -> team -> why it works).
- Tone, sentence length, jargon level, how it positions against competing funded work.

## Candidate sources to source (similar to this proposal)

funded proposals from the SAME agency and program in the proposal's domain (NSF
Award Search, KAKEN database, NSFC funded lists, ERC project pages); agency-released
sample winning proposals; mentors' or collaborators' shared funded applications.
