# PNAS Playbook - Exemplar References

Verified, real exemplar papers in PNAS (and adjacent broad-science venues) that this
manuscript can model on. Each entry says what we borrow: the significance framing, a
display pattern, or the section structure.

RULE: store ONLY real, verifiable papers (DOI where possible). Never fabricate a
citation. Re-verify with `citation-audit` before any of these enters the manuscript
bibliography.

## Search categories

1. Foundational work in the paper's primary classification (the field it advances).
2. Broadly significant PNAS empirics with a comparable method.
3. Cross-disciplinary exemplars spanning the paper's secondary classification.
