# PNAS Exemplars - content to imitate

Store the CONTENT of similar PNAS papers here (PDF, or extracted .md/.txt), so the
writing stages can imitate their **language style and preferences**. This is NOT a
citation list (that is `../references/README.md`); it is a style corpus.

## How to add

- Drop a PDF: `exemplars/<slug>.pdf` (e.g. `author-2023-pnas-<topic>.pdf`).
- Or store extracted text: `exemplars/<slug>.md` (the Significance Statement +
  Abstract + a few Results and Discussion paragraphs are enough to capture style).
- Prefer open-access or author-provided PDFs.

## What to capture per exemplar (feeds `../style-profile.md`)

- Significance Statement arc and exact phrasing (lay-accessible, ~120 words).
- Abstract arc, word budget, and how it states the broadly significant advance.
- How the advance is positioned for a cross-disciplinary readership.
- Discussion structure (advance -> implications -> limitations -> conclusion).
- Tone, sentence length, jargon level, how it frames significance beyond the
  specialty.

## Candidate topics to source (similar to this paper)

broadly significant, reproducible empirics in the paper's domain; cross-disciplinary
exemplars spanning the paper's two classifications; foundational work in both the
primary and secondary fields the manuscript builds on.
