# Nature Portfolio Exemplars - content to imitate

Store the CONTENT of similar Nature Portfolio papers here (PDF, or extracted
.md/.txt), so the writing stages can imitate their **language style and
preferences**. This is NOT a citation list (that is `../references/README.md`); it is
a style corpus.

## How to add

- Drop a PDF: `exemplars/<slug>.pdf` (e.g. `<first-author>-<year>-nmeth-<topic>.pdf`).
- Or store extracted text: `exemplars/<slug>.md` (the summary paragraph + significance
  statement + a few Results and Discussion paragraphs are enough to capture style).
- Prefer open-access or author-provided PDFs.

## What to capture per exemplar (feeds `../style-profile.md`)

- Summary-paragraph / abstract arc and exact phrasing.
- How the broad advance is stated in one non-specialist-readable sentence.
- How benchmarking, validation, and utility are framed.
- Discussion structure (advance -> implications -> limitations).
- Tone, sentence length, jargon level, and which article type the paper used.

## Candidate topics to source (similar to this paper)

broad-significance Nature/Nature Methods/Nature Biotechnology papers in the paper's
domain; method or resource exemplars matching the chosen within-family venue;
foundational papers the manuscript builds on.
