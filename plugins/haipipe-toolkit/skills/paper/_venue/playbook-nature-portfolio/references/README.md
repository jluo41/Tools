# Nature Portfolio Playbook - Sources & Exemplar References

Two things live here: (1) the official Nature Portfolio source pointers that are the
venue's AUTHORITY for the routing and preflight in `../README.md`, and (2) verified,
real exemplar papers this manuscript can model on.

RULE: store ONLY real, verifiable papers (DOI where possible). Never fabricate a
citation. Re-verify with `citation-audit` before any of these enters the manuscript
bibliography.

## Official Nature Portfolio source pointers (venue authority)

Keep these pages as the primary references for fit, article type, and the preflight:

- Nature formatting guide: https://www.nature.com/nature/for-authors/formatting-guide
- Nature editorial criteria and processes: https://www.nature.com/nature/for-authors/editorial-criteria-and-processes
- Nature Methods aims: https://www.nature.com/nmeth/aims
- Nature Methods content: https://www.nature.com/nmeth/content
- Nature Methods reporting standards: https://www.nature.com/nmeth/editorial-policies/reporting-standards
- Nature Methods image integrity: https://www.nature.com/nmeth/editorial-policies/image-integrity
- Nature Methods AI policy: https://www.nature.com/nmeth/editorial-policies/ai
- Nature Methods preprints and conference proceedings: https://www.nature.com/nmeth/editorial-policies/preprints-conference-proceedings
- Nature Biotechnology aims: https://www.nature.com/nbt/aims
- Nature Biotechnology content: https://www.nature.com/nbt/content
- Nature Biotechnology reporting standards: https://www.nature.com/nbt/editorial-policies/reporting-standards

Stable takeaways worth reusing:

- `Nature` expects broad significance and a readable case for non-specialists, not
  just a prettier house style.
- `Nature Methods` cares about technical advance plus validation, comparison,
  reproducibility, and demonstrated biological utility.
- `Nature Biotechnology` needs biotechnology significance to be explicit; many
  platform-scale assets fit better as `Resource` than as a generic full article.
- Reporting standards push the same recurring author burdens: data availability, code
  availability, protocol access, accession IDs, reporting summaries, and clear
  disclosure of restrictions.
- Image-integrity and AI policies are submission-facing requirements, not
  post-acceptance cleanup.
- Preprint, related-manuscript, and conference-proceedings disclosures should be
  checked intentionally before submission.

## Search categories (exemplar citation candidates)

1. Foundational work in the paper's domain (what it builds on).
2. Broad-significance Nature Portfolio papers with a comparable method or resource.
3. Method or resource exemplars matching the chosen within-family venue.
