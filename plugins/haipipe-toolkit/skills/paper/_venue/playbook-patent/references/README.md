# Patent Playbook - Prior-Art References

Verified, real prior art (granted patents, published applications, and academic
papers) that bounds this filing's claims and sets up its Background. Each entry says
what it is used for: the closest reference for novelty, a combination reference for
obviousness, or background context.

RULE: store ONLY real, verifiable references (patent number / DOI where possible).
Never fabricate a patent number or citation; mark uncertain entries with `[VERIFY]`.
Re-verify with `citation-audit` before any of these enters the filing. Patent prior art
includes everything published before the priority date, worldwide, not just patents.

## Search categories

1. Closest single prior-art references (the novelty / anticipation test).
2. Combination references that could be motivated together (the obviousness test).
3. Background-context references in the same technical field (IPC/CPC class).
