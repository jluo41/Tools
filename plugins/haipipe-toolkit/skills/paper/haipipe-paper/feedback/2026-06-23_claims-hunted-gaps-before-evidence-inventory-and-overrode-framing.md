with the same root cause: it theorized the story and HUNTED FOR GAPS before

ROOT CAUSE: the claims stage ran ILLUMINATE/ELICIT-TASTE and GAP-FLAGGING before
EVIDENCE INVENTORY. Correct order is the reverse: (a) inventory ALL existing
judged evidence (sweep probes/ + insights/ + tasks/ first); (b) anchor on the
EXISTING paper framing (title + pitch), don't propose a new primary; (c) only
THEN flag a row as weak/GAP, and only after confirming no existing artifact
covers it; (d) elicit taste only on genuinely-open choices, never manufactured
dilemmas. Net: 5+ correction rounds to reach a ledger that the evidence could
have produced in one pass.

agent default.
