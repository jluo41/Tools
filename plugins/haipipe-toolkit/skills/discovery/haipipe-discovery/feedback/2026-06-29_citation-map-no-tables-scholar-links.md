When discovery produces citation lists or source inventories, NEVER use tables. Each citation/source is its own markdown section (### heading) with bullet-point fields (Key, Journal, Status, Scholar link, Summary). Tables are unreadable when each item carries rich metadata. Always include a clickable Google Scholar search URL so the user can verify and grab BibTeX directly without switching files.
