# UTD24 IS Paper Skills

Claude Code skills for writing Information Systems papers targeting UTD24-ranked journals.

## Skills

| Skill | Purpose |
|-------|---------|
| `is-paper-workflow` | End-to-end IS paper lifecycle — contribution audit, theory, structure, revision |
| `misq-playbook` | MIS Quarterly venue guidance — theory-forward, pluralistic methods, design science |
| `isr-playbook` | ISR venue guidance — theory-driven empirics, causal identification, surveys |
| `ms-is-playbook` | Management Science IS section — economic modeling, platforms, causal econometrics |

## Covered Journals

- MIS Quarterly (MISQ)
- Information Systems Research (ISR)
- Management Science — IS section (MS)

## Installation

```bash
cd Tools && ./install.sh --global --project /path/to/your/repo
```
