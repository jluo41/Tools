Numbered series  (3+ diagrams in one response)
================================================

  When a single answer needs multiple diagrams, prefix each with a
  numbered header so the user can reference them by number.


  Header format
  -------------

  ── [N/TOTAL] Title ──

  - Dashes fill to ~60 chars total width.
  - Omit /TOTAL if the count is not fixed upfront.


  Example  (3-diagram answer)
  ---------------------------

  ── [1/3] Overall Architecture ──────────────────────────────

  +----------+     +------------+     +----------+
  | 📥 Input |---->| 🧠 Model   |---->| 📤 Output|
  +----------+     +------------+     +----------+


  ── [2/3] Folder Layout ─────────────────────────────────────

  code/
  ├── haipipe/    (editable)
  ├── hainn/      (editable)
  └── haifn/      (generated)


  ── [3/3] Stage Status ──────────────────────────────────────

  📥 Load → ⚙️ Clean → 🧠 Train → 📊 Eval → 🚀 Deploy
    ✅        ✅         ⏳          ⬜         ⬜


  Open-ended series  (count not fixed upfront)
  --------------------------------------------

  ── [1] Initial sketch ──────────────────────────────────────

  +-------+      +-------+
  | A     |----->| B     |
  +-------+      +-------+


  ── [2] After review feedback (added retry path) ────────────

  +-------+      +-------+      +-------+
  | A     |----->| B     |<---->| Retry |
  +-------+      +-------+      +-------+


  ── [3] Final ───────────────────────────────────────────────

  +-------+      +-------+      +-------+      +-------+
  | A     |----->| B     |----->| C     |----->| Done  |
  +-------+      +-------+      +-------+      +-------+
