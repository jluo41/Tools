─§ ⚡ Compact form ─────────────────────────────────────────

  🧑  →  Workflow  →  walk phases 00 → 13  →  📄 PDF / patent / talk

         🌿 Nature mindset    discipline · claim-first · venue-fit
         🤖 ARIS mindset      automation · loops · effort levels

         Both compose via WORKFLOW.md routing rules.


─§ 📌 Five truths to keep straight ─────────────────────────

  1. 🎯  Claim before polish.  No auto-loops on unstable claims.

  2. 🖼️  Figure-led results.  figure-planner before scientific-writing.

  3. 🔢  Phase numbers ARE the dependency order — `ls` in order
      walks the workflow.

  4. 🌿🤖  Nature vs ARIS — different mindsets per phase
      (PRINCIPLES.md table). Use BOTH in sequence, not either-or.

  5. 🧾  Skills communicate via plain-text artifact files
      (IDEA_REPORT → EXPERIMENT_PLAN → … → main.pdf).
      See diagram 7-artifacts.txt.
