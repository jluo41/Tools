🚦  State Machine — owned by 🤖 moderator
==========================================

  The moderator is the only state-mutating actor. Every command
  reads + writes {project_dir}/.state.json. Resume-from-anywhere
  is free because state lives only on disk.

─§ 🎯 Verdicts from /sl-validate ───────────────────────────

  Verdict drives whether the BIG loop closes or repeats.
